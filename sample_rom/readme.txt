Sample MegaROM Program
-------------------------------------------------------------------------------

作った MegaROM の動作を確認するためのサンプルプログラムです。
PSG を使って喋ります。( ※一部の PSG互換音源搭載機では音声が出ません。MX-101 等）

音声は、VOICEVOX: 猫使ビィさんを使わせていただきました。
https://voicevox.hiroshiba.jp/

音声の利用に関しては、猫使ビィさんのライセンス規約を守る必要がありますのでご注意ください。
https://nekotukarb.wixsite.com/nekonohako/%E5%88%A9%E7%94%A8%E8%A6%8F%E7%B4%84

動作している動画
https://x.com/thara1129/status/1803029993166921950

エミュレーターで動かす場合は、ASCII16タイプのメガコンを選択してください。
